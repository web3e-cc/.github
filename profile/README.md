<h1 align="center">Web3e</h1>

Open-source integrations for the **Web3e** crypto payment gateway — accept USDT and other crypto in
popular website engines with minimal effort.

| Repo | What |
|---|---|
| [`crypto-gateway-php`](https://github.com/web3e-cc/crypto-gateway-php) | PHP SDK — request signing + webhook verification (the foundation every plugin builds on) |
| [`woocommerce`](https://github.com/web3e-cc/woocommerce) | WooCommerce payment gateway |
| [`whmcs`](https://github.com/web3e-cc/whmcs) | WHMCS payment gateway module |
| [`opencart`](https://github.com/web3e-cc/opencart) | OpenCart payment extension |

Docs: https://docs.web3e.cc · Website: https://web3e.cc
